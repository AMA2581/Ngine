# Dependencies
- GLFW3
- GLM
- SPDLOG
- VULKAN


>just install the    **[LUNARG SDK](https://vulkan.lunarg.com/sdk/home)**






## Build
```cmd
 git clone https://github.com/dadynaderi1/Ngine
 cd Ngine
 cmake -S . -B build
 cmake --build build
```
> ##recommendation 
> open the root folder of project (Ngine) in VSC or CLION
